# SwiftTemplateProject
a template project for ios swift with cocoapods
  

 - includes 4 targets
  * Mock
  * Dev
  * UAT
  * Prod
  
 - includes tools
  * SwiftLint
  * Synx
  
 - includes Pods by default
  * 'SwiftLint'
  * 'Alamofire', '~> 4.0'
  * 'PromiseKit', '~> 4.0'
  * 'PromiseKit/OMGHTTPURLRQ'
  * 'PKHUD', '~> 4.0'
  * 'ReachabilitySwift', '~> 3'
